Mythic+ Leaderboards

Twitter bot written in Python that interacts with Raider.io's API to get a multitude of information on the Mythic+ leaderboards for a specific dungeon in World of Warcraft. Information is then stored in variables, formatted properly, then interacts with Twitter's API to tweet the info. The bot tweets one dungeon's leaderboards once per hour at the same time everyday, 8 times for all 8 dungeons and is scheduled to do that via Task Scheduler.
